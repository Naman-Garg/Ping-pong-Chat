# Ping-pong-Chat
Chat server between 2 person.
Person have to sent 1 message at one time.

# To run:
               
Client.c :-    gcc Client.c -o client
               ./client 127.0.0.1
               
Server.c :-    gcc Server.c -o server
               ./server
