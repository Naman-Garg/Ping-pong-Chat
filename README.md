### Ping-pong-Chat
Chat server between 2 person.
Person have to sent 1 message at one time.

#### To run:
               
Client.c :-   <br> 
gcc Client.c -o client <br>
./client 127.0.0.1
               
Server.c :-   <br>
gcc Server.c -o server <br>
./server
